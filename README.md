# Othello-Agents
This repo is a hub for othello agents that use MinMax &amp; αβ pruning


## Format
```
Student(s) name(s)
 |-----> Package (folder)
         |-----> .java files
 |-----> Presentation (Optional)
 |-----> Report (Optional)
```
